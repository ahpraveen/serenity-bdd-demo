"serenity-bdd cucumber demo" 

Pre-requisites:
JAVA, MAVEN should be installed in your machine.

Setup Instruction:
1/ Go to command prompt

2/ enter the maven command

3/ mvn archetype:generate

4/ choose the project : 1204

5/ Select the stable version

6/ enter the group id and artifact id, hit enter.

7/ Project should be created in your work space.

8/ check serenity.properties file provide the browser details 

9/ mvn verify

10/ should run your test
